# Bluestock Mutual Fund Capstone Analysis

## 📌 Project Overview
This project focuses on analyzing mutual fund performance data to identify high-performing funds, evaluate risk-return relationships, and derive actionable investment insights using data analysis techniques. The goal is to transform raw financial data into meaningful business insights that support better investment decision-making.

---

## 🎯 Problem Statement
Mutual fund investors often struggle to choose funds due to the large number of options and lack of clear comparison metrics. This project addresses that gap by analyzing mutual fund performance, risk levels, and returns to help identify better investment choices based on data-driven insights.

---

## 🎯 Objectives
- Clean and preprocess raw mutual fund dataset
- Analyze fund performance across different categories
- Evaluate risk vs return trade-offs
- Identify top-performing mutual funds
- Build meaningful KPIs for financial analysis
- Generate insights to support investment decisions

---

## 🛠️ Tools & Technologies Used
- Excel / Google Sheets
- Python (Pandas, NumPy) *(if used)*
- SQL *(if used)*
- Power BI / Tableau *(if used)*
- Data Visualization tools

---

## 📂 Dataset Information
The dataset contains mutual fund details including:
- Fund Name  
- Fund Category  
- Net Asset Value (NAV)  
- 1-Year, 3-Year, and 5-Year Returns  
- Risk Levels  
- Expense Ratios *(if available)*  

---

## 📊 Key KPIs & Analysis
- Average returns by fund category  
- Top 10 performing mutual funds  
- Risk vs return comparison  
- Category-wise fund distribution  
- Consistency of fund performance over time  
- Best funds based on long-term stability  

---

## 🔍 Key Insights
- Equity mutual funds generally offer higher returns but come with higher risk  
- Debt funds provide stable returns with lower risk exposure  
- Balanced funds offer an optimal balance between risk and return  
- Certain funds consistently outperform category averages over multiple time periods  
- Risk level is a key factor in long-term investment stability  

---

## 📈 Project Outcome
This analysis helps simplify mutual fund selection by providing clear, data-driven insights into performance and risk. It enables investors to make more informed decisions based on historical trends rather than assumptions.
